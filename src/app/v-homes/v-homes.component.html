For about 1 month, I developed a full-stack real estate web application using Next.js 15 and Firebase (Firestore, Authentication, Storage). This project gave me practical experience working with user authentication (email/password and Google OAuth), role-based access (user/admin), and secure CRUD operations on a Firestore-backed database. I also implemented advanced features like pagination, markdown-rendered property descriptions, image galleries with cloud storage, and favorites functionality gated behind authentication. Admin users can manage property listings through a protected dashboard with image uploading and reordering support. This project significantly deepened my understanding of TypeScript, Zod for schema validation, and scalable component design in a modern React ecosystem.
